# MEAL-RECIPE GENERATOR
This website is one  in which a person inputs the desired food by name and they receive an output listing the ingredients required and the instructions to be followed to make that particular dish.

## Technologies used
This website has been built with the following tools:


- **Node `latest`**

-**Git `latest`**

-**publicAPI**





## Project Setup
You can set up this repository by following this manual

1. Clone this repository
    ```{shell}
    git clone git@github.com:MalualM/project1.git
    ```
2. Use the following public API to fetch data
    ```{shell}
    https://www.themealdb.com/api/json/v1/1/search.php?s=
    ```
3. open the application from your browser
